# Budget App

A personal desktop budgeting application built with React, Vite and IndexedDB (Dexie).

This app is based on the envelope budgeting method:  
each month is divided into budget categories ("envelopes"), where expenses are tracked against planned amounts.

## Features

- Monthly budget management
- Custom budget envelopes
- Expense tracking per envelope
- Real-time remaining balance calculation
- IndexedDB persistence (offline-first)
- Built with React + Vite
- Local-first architecture (no backend)

## Tech Stack

- React
- Vite
- TypeScript
- Dexie (IndexedDB)
- React Router

## Philosophy

This project focuses on simplicity, clarity and control.
No external services, no subscriptions, no cloud dependency — just a clean, fast, local budgeting tool.
